<h1>Promises</h1>

Promises is an app for people who forget all the time! Use this App and get SMS notification 30 minutes prior to your meetings, dates, and etc.....

<h3>Project Requirements</h3>
<ul>
    <li>You must use <a href="www.reactjs.org">React.js</a> and <a href="www.reduxjs.org">Redux</a> in order to manage the state of a frontend application.</li>
</ul>


<h3>Front-end</h3>
<ul>
    <li>Redux</li>
    <li>React</li>
    <li>Bulma CSS framework</li>
</ul>

<h3>Back-end</h3>
<ul>
    <li>ExpressJS</li>
    <li>KnexJS</li>
    <li>Postgresql</li>
</ul>

<p>Utilized Twilio API for sending Text messages 30 minutes prior to your event.</p>
